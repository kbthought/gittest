Explaining Git and GitHub


How would you explain Git and GitHub to a novice? Explain here using HTML.

Git is:

  • a distributed version control system (DVCS)
  • free and open source.
  • designed to handle everything from small to very large projects
  • easy to learn

GitHub is:

  • a collaboration platform built on top of a distributed version control system called Git.
  • also provide features like Issues,Pull Request, Projects, Organizations and Teams to help your team collaborate more effectively
